Ray Tracing

One of the fundamental rendering technique. The images show the effect of having differnet light bounces, from soft shadows to glossy effect and transparency.




Radiosity

A different technique to render light in a scene. The first image represents the form factors in a Cornell Box while the second images shows the radiosity rendering effect. I really like how the light is absorbed creating a nice mixute of colors.




Smoke Rendering

A Smoke-Ring Simulation with the use of Photon Mapping. The photon map was created with the help of an Octree data structure. I bacame so obsesed with this project that I made it into an independent research project with the of my CG professor Barb Cutler (RPI).




Photon Mapping

Photons map for caustic rendering. One of the images shows the distribution of the photons in the scene, while the other image shows the rendering of those photons.




Cloth Simulation

Always have been facinated by cloth. The images represent two different physical simulation engines: a spring-mass cloth system and a grid-based fluid system
